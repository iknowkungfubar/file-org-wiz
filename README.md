# File Org Wiz

An AI-powered file organization system using the PARA + Zettelkasten methodology.

## What's This?

This is a prompt and documentation package to completely reorganize any computer's files using the PARA (Projects, Areas, Resources, Archive) methodology combined with Zettelkasten principles for knowledge management.

## Quick Start

1. **[QUICKSTART.md](./QUICKSTART.md)** - Get started in 5 minutes
2. Install: `pip install flask && python mcp_server.py`
3. Use: Say "organize files at [path]" to your AI
