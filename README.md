# File Org Wiz

An AI-powered file organization system using the PARA + Zettelkasten methodology.

## What's This?

This is a prompt and documentation package to completely reorganize any computer's files using the PARA (Projects, Areas, Resources, Archive) methodology combined with Zettelkasten principles for knowledge management.

## Quick Start

1. **Read the prompt**: `AI_File_Organization_Agent_Prompt.md`
2. **Understand why**: `Why_PARA_Zettelkasten.md`
3. **Give to an AI**: Use the prompt with Claude/GPT/OpenCode to reorganize your files

| File | Purpose |

| `research/` | Original research and documentation |

## Requirements

An AI agent that can:
- Execute bash commands
- Read/write files
- Navigate directory structure

## Methodology

- **PARA**: Projects, Areas, Resources, Archive folder structure
- **Zettelkasten**: Atomic notes with bidirectional linking
- **File naming**: `YYYY-MM-DD__context__description__vNN.ext`

## Output

The system creates:
- PARA folder structure
- Obsidian vault with MOCs
- Naming conventions applied
- Comprehensive user guide
- Backup before changes
