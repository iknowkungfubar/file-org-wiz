# Contributing Guide

Thank you for contributing to file-org-wiz.

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test locally
5. Submit a pull request

## Adding New Installation Guide

Create install_systemname.md with:
- Setup instructions
- How to use
- Troubleshooting

## Code Standards

- Python: PEP 8
- Include docstrings
- Test before submitting

## Testing

Before submitting:
- Run python mcp_server.py --help
- Test health endpoint
- Verify structure creation

## Reporting Issues

Open an issue with clear steps to reproduce.
